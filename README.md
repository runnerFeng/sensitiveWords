# 链接(参数式和行内式)

### 行内式

- [谁敢横刀立马](www.google.com)
- [唯我彭大将军](www.google.com)

### 参数式(先定义，后引用)
   [中国]:www.google.com "中国"
   [法国]:www.baidu.com "法国"
   [planet]:www.earth.com "星球"
- 这里是[中国]，这里是[法国]，我们都在地球这个[planet]上
